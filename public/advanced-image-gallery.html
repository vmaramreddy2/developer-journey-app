Advanced Image Gallery

using jQuery, Swiper.js and fullscreen repository


Swipe or click an image in the bottom row, or click one of the pagination bullets to navigate. Click the larger image to open the larger image gallery. In the larger image gallery: click an image to zoom in or out. Click the icons in the navigation bar to zoom, close or even go fullscreen.




Note (currently disabled): you can also use your keyboard (arrow keys, spacebar, enter and escape) to navigate, zoom and open/close the larger image gallery.


External sources:

  • swiper.min.css
  • jquery 3.1.0
  • swiper.min.js



Known issues:

if you have found a way to fix some errors, please link your forked pen in a comment.


Bug #1 - after sliding using the mouse, keyboard navigation might not working correctly (fix: set keyboardControl to false).

Bug #2 - when navigating and zooming in the larger image gallery, the keyboard navigation doesn't work correctly with the smaller sliders (fix: set keyboardControl to false).

Bug #3 - when setting loop to true, inside the swiper options, the swiper.params.control won't be able to synchronize with the other swipers.

Bug #4 - Not fully supported in IE.

Bug #5 - When you click on a bullet pagination twice in a short time, the slider will freeze and needs a swipe before it works normal again (fix: set clickablePagination to false).
